LIBS

Laser Induced Breakdown Spectroscopy(LIBS) is a versatile tool for quantitatively and qualitatively analyzing a sample using laser ablation. It is a rapid and portable technique that requires no sample preparation. In LIBS the optical emissions from the plasma plume produced by the pulsed laser ablation of the sample surface are recorded and compared to a database of atomic emission spectra of known materials.

Input Parameters

Spectral Data Input : openLIBS accepts the spectral data from the user in CSV format which contains a 2D array with the first column representing the wavelength (nm) and the second column representing the corresponding intensities of atomic emissions from the sample. Header should not be included. An image of a sample spectral data is provided for reference.
Left Wavelength Limit : The minimum value of the wavelength range across which the data needs to be analyzed.
Right Wavelength Limit : The maximum value of the wavelength range across which the data needs to be analyzed.
Required Minimum Intensity : The minimum intensity required for a peak to be considered for analysis.
Allowed Left Error : The minimum difference in wavelength (nm) allowed for a peak to be considered a match. Input the magnitude without sign.
Allowed Right Error : The maximum difference in wavelength (nm) allowed for a peak to be considered a match.
Match Threshold : The minimum number of peaks that must match the standard peaks for an element to be considered a match.

Results and Interpretation

  • List of the detected elements.
  • A plot showing the detected peaks for each element.
  • An array corresponding to each detected element that includes the observed peak value in the spectrum, the intensity or strength of the peak at the given wavelength, the standard wavelength value for the peak, and the ionization state of the element corresponding to the peak.
