About PAGE

The openLIBS website provides a convenient environment for the Laser Induced Breakdown Spectroscopy (LIBS) data analysis to obtain qualitative information about the sample. The website uses the latest version of the OpenLIBS package as a back end enabling direct and quick access for LIBS analysis.
